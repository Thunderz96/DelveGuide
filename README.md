# DelveGuide — WoW Midnight Addon

A reference guide for Midnight Delves — rankings, curios, loot tables, and a completion history tracker.

## Features

- Delve listings by zone with difficulty rankings
- Curio database with slot and effect details
- Loot table reference
- Completion history — logs each delve run with timestamp
- Active variant scanner using C_AreaPoiInfo
- Minimap button — draggable, left-click to toggle

## Usage

```
/dg            Open / close
/dg scan       Rescan active delve variants
/dg map        Toggle world map
/dg font 1.2   Scale the UI text (0.6–2.0)
/dg hide       Hide the window
```
